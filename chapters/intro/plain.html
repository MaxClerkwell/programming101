How to use this slide-deck

This slide-deck comes with three major features that I use to my convenience:

  • Internationalization
    • i18n is used and the language can be set by the URL parameter language to en or de
  • Themes
    • Since I use this presentation for my lectures at different universities, I use the theme parameter to customize the layout and certain chapters for HSBO, THGA and RUB
  • Dynamic Chapter Selection
    • The chapter parameter can be used to select certain chapters dynamically. This makes it easier to include sub-parts of the content as iFrames in moodle and other platforms
